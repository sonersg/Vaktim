## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```

3. Useful Commands

   ```bash
   npx expo prebuild
   npx expo run:android
   npx expo run:android --device
   npx expo run:android --variant Release --device
   npx expo prebuild --platform ios

   cd android && ./gradlew clean
   cd ..
   ```
